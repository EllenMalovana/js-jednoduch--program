# Jednoduchý JS program – práce se známkami

## Co program dělá
- obsahuje pole známek  
- počítá průměr  
- hledá nejlepší a nejhorší známku  
- filtruje známky lepší než 3  
- vypisuje výsledek do konzole  

## Jak spustit
1. Otevři terminál ve VS Code  
2. Spusť: `node script.js`  

## Příklad vstupu
const grades = [1, 2, 3, 1, 4, 2, 1];

## Příklad výstupu
Známky: [1, 2, 3, 1, 4, 2, 1]  
Průměr: 2  
Nejlepší známka: 1  
Nejhorší známka: 4  
Známky lepší než 3: [1, 2, 1, 2, 1]
